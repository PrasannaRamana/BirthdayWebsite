Place your birthday song here.

1. Add your music file (for example: birthday-song.mp3)
2. Open js/script.js
3. Update BIRTHDAY_CONFIG.music.src to match your file name

Example:
  music: {
    src: "assets/birthday-song.mp3",
    volume: 0.45,
    loop: true
  }

The song will auto-play when "Begin the Journey" is clicked.
Use the Music button (top-right) to mute/unmute anytime.
