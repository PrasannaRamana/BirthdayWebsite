# Birthday Website

An interactive birthday surprise site for Bhavana — built with HTML, CSS, and JavaScript.

## Open locally

Open `index.html` in your browser, or use a simple local server.

## Customize

Edit `js/script.js`:

- **Name & birthday:** `BIRTHDAY_CONFIG.name` and `BIRTHDAY_CONFIG.birthday`
- **Music:** add `assets/birthday-song.mp3` and set `BIRTHDAY_CONFIG.music.src`
- **Photos:** update `BIRTHDAY_CONFIG.photos` (default: `assets/photo1.png` … `photo6.png`)

## Live site

Hosted on GitHub Pages from the `main` branch.
